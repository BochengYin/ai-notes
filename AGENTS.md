# Agent Instructions

This is the public AI Notes repository.

## Content Rules

- Public content must be English and publish-ready.
- Do not add raw private notes, local file paths, private project context, secrets, or full third-party course dumps.
- Reusable files belong in `artifacts/`.
- Website content belongs in `content/`.
- Authoring templates belong in `templates/`.

## Promotion Rule

Only promote content from private notes when the user explicitly approves that specific item for publication.

## Verification

Before finishing changes, run:

```bash
npm run build
```
