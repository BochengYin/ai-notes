# AI Notes

Public AI Field Notes by Bocheng Yin.

This repository is a public notebook for practical AI building workflows, reusable agent components, and small experiments. It is intentionally not a raw source archive. Content here should be English, digested, publish-ready, and free of private local references.

## Structure

- `content/field-notes/`: practical notes and cases.
- `content/components/`: reusable agent components such as skills, commands, agents, hooks, templates, decision cards, and kits.
- `content/experiments/`: short experiment writeups.
- `artifacts/`: reusable files that readers or agents can copy.
- `templates/`: authoring templates.
- `src/`: Astro site shell.
